# Como Iniciar!

primeiramente, **instale o módulo do discord.py.** Instale utilizando o seguinte comando no cmd:

```
python -m pip install discord.py
```

Após isso, **importe o modulo do discord na sua instância** escrevendo o seguinte:

```python
import discord
```
